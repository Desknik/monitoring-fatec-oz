# Ferramentas utilizadas nas aulas

Aqui você encontrará todas as ferramentas que utilizamos nas aulas, desde editores de texto até compiladores de código.


## Editor de código
[Programiz](https://www.programiz.com/python-programming/online-compiler/) Programiz.com oferece um compilador on-line amigável que permite escrever e executar
código em diversas linguagens de programação on-line.

## Editor de Markdown
[StackEdit](https://stackedit.io/) Editor de Markdown completo e de código aberto baseado no PageDown, na biblioteca
**Markdown** usada pelo Stack Overflow e nos outros sites do Stack Exchange.


## Quadro branco
[Excalidraw](https://excalidraw.com/) Uma ferramenta de quadro branco colaborativo virtual que permite que você
facilmente esboce diagramas que têm uma sensação desenhada à mão pra eles.

## Protótipos, Diagramas e mais 
[Figma](https://www.figma.com/) Figma é uma ferramenta usada para projetar, prototipar, desenvolver e testar produtos em um único
espaço. Ele também oferece FigJam, um quadro branco online para alinhar equipes, e Dev Mode...
